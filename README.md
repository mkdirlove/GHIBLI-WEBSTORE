# GHIBLI-WEBSTORE

Our final project in Web Systems and Technology

* CSS
* Bootstrap
* JavaScript
* Python

# INSTALLATION & USAGE

      $ git clone https://github.com/mkdirlove/GHIBLI-WEBSTORE.git
      $ cd GHIBLI-WEBSTORE
      $ python3 -m pip install -r requirements.txt
      $ python3 server.py
